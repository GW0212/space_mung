사용 방법
1. worlds 폴더 안의 각 우주 폴더에 아래 파일 2개를 넣습니다.
   - video.mp4
   - poster.jpg
2. 파일명은 반드시 위 이름으로 맞춥니다.
3. GitHub Pages에 그대로 업로드하면 됩니다.

권장 사양
- 영상 길이: 10~20초
- 포맷: mp4 (H.264)
- 해상도: 1920x1080 또는 1280x720
- 루프가 자연스럽게 이어지는 영상 추천
- poster.jpg는 첫 화면 검은 화면 방지용

우주를 더 추가하고 싶다면
- worlds.json에 항목 추가
- worlds 안에 같은 이름 폴더 생성
- 그 안에 video.mp4 / poster.jpg 추가
